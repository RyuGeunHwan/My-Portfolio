피트니스 페이지 제작


* 해당 주제를 선정한 이유?
웨이트 트레이닝에 관심이 있어, 수업 중에 배운 HTML과 CSS를 이용해서 싱글페이지를 제작해보았습니다. 개발기간은 약 2일이 소요되었고,
깃허브에서 소스를 확인하실 수 있습니다.
github source

* 느낀점
Semantic(footer, header, body, section 등) UI를 기초로 HTML 레이아웃을 구성하였고,
수업시간에 배운 Flex와 Grid를 중점으로 CSS를 작성하였습니다.
또, 유튜브의 영상을 삽입하여 구현하여 다양하게 활용하는 방법을 알게 되었고,
이미지를 슬라이드로 표현하는 방법과 카카오맵 API를 카카오맵을 구현하는 방법을 터득하였습니다.

* Flex와 Grid의 차이를 정확히 느끼지 못해 조사를 해본 결과
1. Flex는 1차원으로 수평, 수직 영역 중 하나의 방향으로만 레이아웃을 나눌 수 있음.
2. Grid는 2차원으로 레이아웃을 수평 수직으로 동시에 영역을 나눌 수 있음.

아쉬운점
